## 📌 Overview

**Sound Sweet** is a modern web-based audio editor built to make audio editing simple, enjoyable, and accessible. Unlike traditional audio software with complex interfaces, Sound Sweet provides a colorful and intuitive experience with interactive controls, animated visualizations, and AI-powered audio enhancement tools. :contentReference[oaicite:0]{index=0}

Whether you're creating podcasts, social media content, music clips, or voice recordings, Sound Sweet helps you achieve professional-quality audio without the learning curve. :contentReference[oaicite:1]{index=1}

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/sound-sweet.git
```

### Navigate to Project

```bash
cd sound-sweet
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
